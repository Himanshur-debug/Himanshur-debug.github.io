  • TIC-TAC-TOE GAME
    • Technologies Used: C++
    • Description: Developed a console-based Tic-Tac-Toe game in C++ that allows two players to compete against each other. The game features a simple user interface with a 3x3 board and supports basic gameplay mechanics including move validation, win detection, and tie handling.
    • Key Features:
      • Player Input: Accepts player names and allows each player to take turns entering their moves by selecting a number corresponding to a cell on the 3x3 grid.
      • Board Display: Updates and displays the current state of the board after each move, showing Xs and Os as they are placed.
      • Move Validation: Ensures that moves are valid, preventing players from placing their tokens in already occupied cells and handling invalid inputs.
      • Game Status Detection: Checks for a win condition or a draw after each move. A win is detected if a player places three of their tokens in a row, column, or diagonal. A draw is declared if the board is full and there is no winner.
      • Game End: Announces the result of the game, displaying which player won or if the game ended in a draw.
